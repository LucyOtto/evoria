# Evoria

Evoria est une application web de planification et de suivi d’événements, pensée pour remplacer Excel par une interface plus visuelle, intuitive et centralisée.

## Objectifs
- Centraliser les informations d’un événement
- Suivre les tâches et le rétroplanning
- Gérer l’équipe, les ressources et le budget
- Offrir une interface simple pour les étudiants, associations et petites structures

## Stack technique
- Frontend : React + TypeScript + Tailwind
- Backend : FastAPI
- Base de données : SQLite

## Fonctionnalités prévues
- Gestion des événements
- Gestion des tâches
- Rétroplanning visuel
- Tableau de bord
- Carnet de bord
- Budget et ressources
